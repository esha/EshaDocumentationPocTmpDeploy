NutrientVector Class

Namespace: Esha.DomainModel
Assembly: Esha.DomainModel (in Esha.DomainModel.dll)
Represents an indexed sequence of quantities of Nutrients. Vector operations (scalar multiplication, vector addition) can be performed with the same meanings as generalized vector operations. Values are stored sparsely.
